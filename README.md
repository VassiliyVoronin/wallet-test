## Тестовое задание

### Описание
Прочитайте внимательно задание. На https://github.com прошу написать код для реализации задачи

Описание задачи: вам необходимо написать код, по которому происходит зачисление и списание крипто-баланса пользователя, с учетом рисков

Цель:
Разработать модуль учета крипто-баланса пользователя, который позволяет корректно и безопасно:
- зачислять средства
- списывать средства (выводы, платежи, комиссии),с учетом специфики блокчейна, асинхронности и рисков

Используйте Стек технологий PHP+Laravel

После того как написали код, приложите ссылку с https://github.com , убедитесь, что открыли доступ на просмотр.

P.S. Не усложняйте задание, оно максимально простое, вам только необходимо описать зачисление и списание ЛК крипто-баланса пользователя, с учетом рисков.

Обычно задание занимает 30-60 минут

### Реализация
Сущности:
- user: пользователь
- wallet: кошелек пользователя. Имеет 2 свойства: available_sum(доступная сумма средств) и frozen_sum(замороженная сумма средств)
- transactions: список транзакций. Имеет 2 типа(deposit/withdraw) и 3 статуса(pending/confirmed/failed).

App\Application\DepositService - сервис приема средств: обновляет доступную сумму баланса и создает транзакцию.

App\Application\WithdrawService - сервис списания средств, имеет 3 метода:
- withdraw: замораживает списываемую сумму;
- confirm: подтверждение и списание суммы;
- fail: ошибка, возврат замороженной суммы в доступную;

При любой транзакции кошелек блокируется на изменение.

На все методы реализованы тесты:
```bash
php artisan test
```
